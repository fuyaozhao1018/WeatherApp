## WeatherApp

This is a web app that allows users to check weather details of cities.

## Functions 
- Searching for cities
- Save City
- Remove City

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
